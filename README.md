# SSTO Feasibility Analysis

Computational investigation of Single-Stage-to-Orbit
(SSTO) feasibility using chemical and nuclear thermal
propulsion.

## Key Result

At ε = 0.10:

- Chemical SSTO (Isp = 450 s): λ ≈ 2.1%
- Nuclear SSTO (Isp = 800 s): λ ≈ 22.3%
